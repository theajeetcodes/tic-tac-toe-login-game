# Tic Tac Toe Login Game

## Project Overview
This is a simple **Tic Tac Toe game** with a login page built using **HTML, CSS, and JavaScript**.  
The project demonstrates basic frontend skills including **DOM manipulation, event handling**, and **GitHub Pages deployment**.
